# Audio sticks

Разработано в 2024

**Идея**: голосовые стикеры в телеграм, голосовые стикер-паки

**Функционал** 
1. создание своего стикер-пака: бота
2. в боте стикер-паке вы можете добавлять/редактировать голосовые стикеры
3. отправляйте голосовые стикеры через inline режим вашего бота
4. управляйте своими стикер-паками через главного бота
